# REFLECT – Emotion Aware Journal

An AI-powered journaling application that analyzes emotions from journal entries and provides insights.

## Features
- Write journal entries
- Emotion detection using NLP
- Emotion analytics dashboard
- AI mood insights

## Tech Stack
Frontend: React + Tailwind CSS  
Backend: FastAPI  
NLP Model: DistilRoBERTa (Transformers)  
Database: SQLite
